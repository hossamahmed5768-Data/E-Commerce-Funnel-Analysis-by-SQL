# E-commerce Funnel Analysis (SQL Project)

## Objective
Analyze user behavior through the sales funnel and measure conversion rates and revenue performance.

## Dataset
- Source: e-commerce event tracking data
- Columns: user_id, event_type, event_date, traffic_source, amount

## Analysis Performed
- Funnel conversion rates (page view → purchase)
- Funnel by traffic source
- Time to conversion analysis
- Revenue metrics (AOV, revenue per visitor, etc.)

## Tools Used
- SQL (MySQL)
